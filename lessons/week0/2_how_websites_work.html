Week 0: How Websites Work

Ever wondered what happens when you type a website address and hit Enter? Let's break it down.

Core Concepts

  • Client: This is your web browser (e.g., Chrome, Firefox). It *requests* information.
  • Server: This is a powerful computer somewhere else in the world that stores the website's files. It *serves* information.
  • When you go to a website, your browser (the client) sends a request over the internet to the website's server. The server finds the right files (HTML, CSS, JS) and sends them back to your browser. Your browser then reads those files to display the page.

The Three Pillars of the Web

Think of it like building a house:

  • HTML: The Nouns (the structure, the "bones").
  • CSS: The Adjectives (the styling, the "decoration").
  • JavaScript (JS): The Verbs (the interactivity, the "electricity").
