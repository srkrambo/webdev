Week 1: Images & File Paths

Images bring life to a webpage. The <img> tag lets you embed images, but to do so, you need to understand how to tell the browser where to find the image file using file paths.

Core Concepts

  • The Image Tag: The <img> tag is a self-closing tag used to display an image.
  • The src Attribute: The "source" attribute is mandatory. It contains the path or URL to the image you want to display.
  • The alt Attribute: The "alternative text" attribute is also mandatory for accessibility. It provides a text description of the image for screen readers or if the image fails to load.
  • File Paths:
    • Absolute Paths: A full URL to an image on another website (e.g., https://.../image.jpg).
    • Relative Paths: A path to an image relative to your current HTML file. For example, src="images/logo.png" tells the browser to look for an `images` folder in the same directory as the HTML file, and find `logo.png` inside it.
  • Favicons: A favicon is a small icon that appears in the browser tab. You add it in the <head> section using <link rel="icon" href="path/to/favicon.ico">.
