Week 1: Document Structure

Every HTML page is built on a foundation of essential tags. These tags don't display content directly, but they tell the browser what kind of document it is and how to start rendering it.

Core Concepts

  • <!DOCTYPE html>: This is the very first thing in your document. It tells the browser that the page is written in modern HTML (HTML5).
  • <html>: This is the root element that wraps all the content on the entire page.
  • <head>: This section contains meta-information about the page that isn't displayed directly. This includes the page title, character set, and links to stylesheets.
  • <body>: This section contains all the visible content of the web page, like headings, paragraphs, images, and links.
  • <title>: Sets the title of the page, which appears in the browser tab.
  • <meta charset="UTF-8">: Ensures the browser interprets characters correctly.
