Week 1: Links & Attributes

Links, or hyperlinks, are what make the web a "web." They connect pages and resources, allowing users to navigate between them. HTML tags can also have attributes, which provide extra information about an element.

Core Concepts

  • The Anchor Tag: The <a> tag (for "anchor") creates a hyperlink. The text between the opening and closing tags becomes the clickable link.
  • The href Attribute: This is the most important attribute. It specifies the destination URL (e.g., href="https://www.google.com").
  • The target Attribute: Controls where the link opens. Use target="_blank" to open the link in a new tab, which is common for external sites.
  • The title Attribute: Provides extra information about the link, which often appears as a tooltip when you hover over it.
  • Link Types: You can link to external websites, other pages on your own site (e.g., href="/about.html"), sections on the same page (href="#contact"), or even trigger an email client (href="mailto:test@example.com").
