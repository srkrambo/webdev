Week 0: What is Code?

Welcome to your first step! Let's demystify what "code" actually is.

Core Concepts

  • Code is a set of instructions that tells a computer exactly what to do.
  • Think of it like a recipe: a list of steps for the computer to follow to achieve a result.
  • Unlike human languages, programming languages are very strict and have no ambiguity.
